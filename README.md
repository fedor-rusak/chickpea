# Chickpea

Media runtime in C++/Javascript. MIT License.

# Supported platforms

 * Windows 7, 8 (32bit) (requires Visual Studio 2012+)
 * Ubuntu 12.04, 14.04 (64 bit) (requires manual dependency resolving)
